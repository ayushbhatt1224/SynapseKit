# SynapseKit Discord Custom Commands

Draft for issue #396.

## Tool
Use **Carl-bot** (Custom Commands).

Carl-bot dashboard: https://carl.gg

---

## Commands to set up

| Command | Response |
|---|---|
| `/docs` | https://synapsekit.github.io/synapsekit-docs/ |
| `/docs rag` | https://synapsekit.github.io/synapsekit-docs/docs/rag/pipeline |
| `/docs agents` | https://synapsekit.github.io/synapsekit-docs/docs/agents/overview |
| `/docs graph` | https://synapsekit.github.io/synapsekit-docs/docs/graph/overview |
| `/docs providers` | https://synapsekit.github.io/synapsekit-docs/docs/llms/overview |
| `/docs install` | https://synapsekit.github.io/synapsekit-docs/docs/getting-started/installation |
| `/docs quickstart` | https://synapsekit.github.io/synapsekit-docs/docs/getting-started/quickstart |
| `/changelog` | https://synapsekit.github.io/synapsekit-docs/docs/changelog |
| `/pypi` | https://pypi.org/project/synapsekit/ |
| `/github` | https://github.com/SynapseKit/SynapseKit |

---

## Setup path

Carl-bot dashboard → Custom Commands → create each command above. Takes ~15 mins.

These work in all channels so anyone can type `/docs rag` and get the link instantly.
